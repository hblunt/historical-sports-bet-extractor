# Sports Bet Extractor v1.0

A simple desktop app that fetches Australian racing results and odds from the TAB API and exports them to Excel.

## Features

- Fetch historical results for **Thoroughbreds, Harness, and Greyhounds**
- Filter by **state** (QLD, NSW, VIC, SA, WA, TAS, ACT, NT) and **date range**
- Choose what to export: **winners only, top 4 placegetters, or all runners**
- Pick which **columns** to include (date, venue, race, distance, runner, jockey, trainer, odds, etc.)
- Exports to a formatted **.xlsx** file
- Works on **Windows and Mac**

## Download

Grab the latest build from the [Actions tab](../../actions) — download the artifact for your OS and run it. No Python install needed.
